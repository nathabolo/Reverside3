# Time sheet and leave form Example with Spring Security, Spring Boot, Spring Data JPA, HSQL, JSP/Angular 4

## Guide
www.reverside.co.za

## Prerequisites
- JDK 1.7 or later
- Maven 3 or later

## Stack
- Spring Security
- Spring Boot
- Spring Data JPA
- Maven
- JSP/Angular 4
- HSQL

## Run
```mvn clean spring-boot:run```
